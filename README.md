RestoSystem El Alto 2026 🍽️
Sistema Integrado TPS-MIS-DSS para Restaurantes
Universidad Pública de El Alto - UPEA 2026

📋 Descripción
Sistema web que integra los 3 niveles de sistemas de información:

TPS: Registro de pedidos en tiempo real
MIS: Dashboard con KPIs y gráficas Chart.js
DSS: Alertas automáticas de stock bajo
Patrón Observer: Módulo Cocina actualiza en tiempo real
Arquitectura N-Tier: 4 capas separadas
🛠️ Tecnologías
Backend: Python 3.11, Flask 2.3, SQLAlchemy 2.0
Frontend: HTML5, Bootstrap 5.3, Chart.js 4.4
Base de Datos: SQLite 3
🚀 Instalación
bash
1. pip install flask flask-sqlalchemy
2. py app.py
3. Abrir http://127.0.0.1:5000
